## 01. 구현 기능

### 🚗 PDW / PAS — 주차 거리 경고 시스템

초음파 센서로 장애물을 감지하여 5단계 경고를 제공합니다.

| 단계 | 거리 | 동작 |
|------|------|------|
| 1단계 | 21cm 초과 | 경고음 없음 |
| 2단계 | 15~21cm | 간헐적 경고음 |
| 3단계 | 9~15cm | 일반 경고음 |
| 4단계 | 3~9cm | 빠른 경고음 |
| 5단계 | 3cm 이하 | 연속음 + PCA |

---

### ⚡ PCA — 주차 충돌방지 보조

충돌 임계치 도달 시 운전자 조작과 무관하게 모터를 즉각 차단합니다.

1. 5단계 진입 시 제동 트리거
2. 제어 우선순위 재조정 (Override)
3. 모터 드라이버 전자식 브레이크
4. 완전 정지 확인 후 제어권 회복

---

### 🅿️ APA — 자율 주차 시스템

궤적을 자율 생성하여 평행 및 T자 주차를 수행합니다.

- **평행 주차** — 4단계: 진입 대기 → 1차 후진 → 역회전 → 정렬
- **T자 주차** — 3단계: 변곡점 계산 → 후진 진입 → 수평 정렬
- **기반**: 엔코더 이동 거리 + 서보 조향각 PID 제어
