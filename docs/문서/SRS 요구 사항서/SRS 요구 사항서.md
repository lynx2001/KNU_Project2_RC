# SRS 요구 사항서


# System Requirements Specification


###  Functional Requirements 기능적 요구사항


시스템이 '무엇을' 해야 하는가에 대한 명세


FR


### Non-Functional Requirements 비기능적 요구사항 


시스템이 '어떻게' 동작해야 하는가(성능, 안전성, 환경)에 대한 명세


NF


# Requirements Traceability Matrix


**시스템 요구사항 추적성 매트릭스**

- 요구사항이 어떤 하드웨어/소프트웨어 컴포넌트로 구현되는가
- 어떤 QA를 통해 검증되는가

⇒ 누락된 기능이 없는지 확인하는 용도


RTM
